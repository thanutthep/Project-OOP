# อธิบายเว็บ ร้านเครื่องดื่ม OOP (สำหรับใส่สไลด์)

---

| ขั้น | ทำอะไร | ข้อความสไลด์สั้นๆ |
|-----|--------|-------------------|
| **1** | **เปิด Demo** — ให้ดูเว็บจริง เลือกเมนู กดปุ่ม | "นี่คือเว็บที่เราทำ: เลือกเครื่องดื่ม แล้วกดดูรายละเอียด" |
| **2** | **บอกว่าใช้อะไร** — Python รันในเบราว์เซอร์ได้เพราะ PyScript | "ใช้ PyScript ให้ Python รันบนเบราว์เซอร์ ไม่ต้องมีเซิร์ฟเวอร์" |
| **3** | **อธิบาย OOP แบบภาพ** — คลาสแม่หนึ่งอัน คลาสลูกสามอัน | "ด้านหลังใช้ OOP: มีคลาสแม่ Drink แล้วมีคลาสลูก Coffee, MilkTea, Smoothie" |
| **4** | **แสดงตาราง/รูป** — ตาราง Drink / Coffee / MilkTea / Smoothie | ใช้ตารางในหัวข้อ 3 ด้านล่าง |

**เคล็ดลับ:** เริ่มจาก "ให้ดูของจริง" แล้วค่อยอธิบายแนวคิด จะฟังแล้วเข้าใจง่ายกว่าอธิบายโค้ดก่อน

---

## 1. เว็บนี้ทำอะไร

- ให้ผู้ใช้ **เลือกเมนูเครื่องดื่ม** จาก dropdown (กาแฟ / ชานมไข่มุก / สมูทตี้)
- กดปุ่ม **ดูรายละเอียดเครื่องดื่ม** แล้วจะโผล่ **กล่องแจ้งเตือน (alert)** บอกชื่อที่เลือก และแสดง **ข้อความอธิบาย** (ชื่อ เมนู ราคา ระดับความหวาน) ในกล่องด้านล่าง

---

## 2. ทำงานยังไง (ภาพรวม)

1. ใช้ **PyScript** ให้ Python รันบนเบราว์เซอร์
2. เมื่อกดปุ่ม โปรแกรมจะดูว่าผู้ใช้เลือกเมนูอะไร แล้ว **สร้าง object จากคลาสลูก** (Coffee / MilkTea / Smoothie) ที่สืบทอดจากคลาสแม่ Drink
3. เรียก **method** `get_description()` ของ object นั้น แล้วเอาไปแสดงในหน้าเว็บ และใช้ `window.alert()` แจ้งชื่อเมนูที่เลือก

---

## 3. โครงสร้าง OOP (คลาสแม่–คลาสลูก)

| คลาส | ความหมาย |
|------|----------|
| **Drink** (คลาสแม่) | เครื่องดื่มทั่วไป มี attribute ชื่อ ราคา ระดับความหวาน และ method get_description() |
| **Coffee** (คลาสลูก) | กาแฟเย็น ราคา 45 บาท หวานน้อย |
| **MilkTea** (คลาสลูก) | ชานมไข่มุก ราคา 55 บาท หวานปานกลาง |
| **Smoothie** (คลาสลูก) | สมูทตี้ผลไม้รวม ราคา 65 บาท หวานน้อย–ปานกลาง |

คลาสลูกใช้ `super().__init__(...)` เรียก constructor ของคลาสแม่ แล้วส่งค่าเข้าไป

---

## 4. สิ่งที่เว็บแสดง

- **กล่องแจ้งเตือน (alert)** = ชื่อเมนูที่เลือก (จาก attribute `drink.name`)
- **กล่อง output ด้านล่าง** = ข้อความจาก `get_description()` (ชื่อ ราคา ระดับความหวาน)

---

## 5. สรุปหนึ่งบรรทัด (สำหรับสไลด์)

**เว็บร้านเครื่องดื่ม OOP ใช้ PyScript ให้ผู้ใช้เลือกเมนู แล้วสร้าง object จากคลาสแม่ Drink และคลาสลูก Coffee / MilkTea / Smoothie แล้วแสดงรายละเอียดด้วย method get_description() และ alert**
